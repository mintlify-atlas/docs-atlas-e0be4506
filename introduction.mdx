## What is SUGO Front?

SUGO Front is a web-based management system built with React and TypeScript that streamlines dispatch (despacho) and reception (recepción) operations. The application provides intuitive form interfaces for tracking vehicle movements and managing module assignments within your organization.

## Technology Stack

SUGO Front is built with cutting-edge technologies:

- **React 19.2.0** - Latest React version with modern features
- **TypeScript 5.9.3** - Type-safe development experience
- **Vite 7.3.1** - Lightning-fast build tool with HMR
- **PrimeReact 10.9.7** - Rich UI component library
- **Bootstrap 5.3.8** - Responsive CSS framework
- **React Router 7.13.1** - Client-side routing
- **FontAwesome** - Comprehensive icon library

## Project Architecture

The application follows a modular structure:

```
src/
├── App.tsx              # Main app component with routing
├── main.tsx             # Application entry point
├── components/          # Reusable UI components
│   ├── Header.tsx
│   └── Sugo_main.tsx
├── Despacho/           # Dispatch module
│   └── FormularioDespacho.tsx
├── Recepcion/          # Reception module
│   └── FormularioRecepcion.tsx
└── hooks/              # Custom React hooks
    ├── useModulos.ts
    └── useHook.ts
```

## Use Cases

SUGO Front is ideal for:

- Logistics companies managing vehicle dispatch and reception
- Warehouse operations tracking incoming and outgoing materials
- Transportation hubs coordinating vehicle movements
- Organizations needing centralized module assignment systems
