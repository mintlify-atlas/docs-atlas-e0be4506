## PrimeReact Theme Setup

SUGO Front uses the **Saga Blue** theme from PrimeReact, configured in `main.tsx`:

```tsx src/main.tsx
import "primereact/resources/themes/saga-blue/theme.css";
import "primereact/resources/primereact.min.css";
import "primeicons/primeicons.css";
```

### Version Information

From `package.json`:

```json
{
  "primereact": "^10.9.7",
  "primeicons": "^7.0.0"
}
```

## Component Examples

### Menubar with Custom Styling

<CodeGroup>
```tsx src/components/Header.tsx
import { Menubar } from "primereact/menubar";
import { useNavigate } from "react-router-dom";

export const Header = () => {
  const navigate = useNavigate();

  const items = [
    {
      label: "Despacho",
      icon: "pi pi-fw pi-home",
      command: () => navigate("/despacho"),
    },
    {
      label: "Recepcion",
      icon: "pi pi-fw pi-calendar",
      command: () => navigate("/recepcion"),
    },
  ];

  const end = (
    <div style={{ display: "flex", alignItems: "center", gap: "1.5rem" }}>
      <i className="pi pi-user" style={{ fontSize: "1.1rem" }}></i>
    </div>
  );

  return <Menubar model={items} className="menuBar" end={end} />;
};
```

```css App.css
.menuBar {
  margin: 0.8rem;
  box-shadow: 0.5rem 0.5rem 1rem rgba(129, 128, 128, 0.2);
}

.menuBar .p-menubar-root-list {
  justify-content: center !important;
  margin-top: 0.4rem;
}
```
</CodeGroup>

### InputText with Float Label

```tsx src/Despacho/FormularioDespacho.tsx

<span className="p-float-label">
  <InputText className="select" />
  <label htmlFor="username">Economico</label>

```

<Note>
The `p-float-label` class is a PrimeReact utility that creates animated floating labels when the input is focused or has a value.
</Note>

## Component Customization

### Overriding PrimeReact Styles

To customize PrimeReact components, target their specific CSS classes:

```css App.css
/* Target specific PrimeReact component classes */
.menuBar .p-menuitem-link {
  margin-right: 1rem;
  margin-left: -1rem;
}

.menuBar .p-menubar-root-list {
  justify-content: center !important;
  margin-top: 0.4rem;
}

.p-tabview-nav .p-tabview-nav-link {
  text-decoration: none !important;
}
```

<Warning>
Use `!important` when overriding PrimeReact's default styles to ensure your custom styles take precedence.
</Warning>

### Theme Customization with CSS Variables

PrimeReact themes can be customized using CSS variables:

```css
:root {
  --primary-color: #f1d262;
  --primary-color-text: #19160b;
  --surface-border: #7c6c34;
}
```

## Best Practices

<Steps>
  <Step title="Use semantic severity levels">
    Leverage Button severity props (`success`, `danger`, `info`, `warning`) for consistent UI feedback.
  </Step>
  
  <Step title="Combine with float labels">
    Use `p-float-label` wrapper for better UX with InputText and Dropdown components.
  </Step>
  
  <Step title="Leverage PrimeIcons">
    Use PrimeIcons instead of custom icons for consistency with the PrimeReact theme.
  </Step>
  
  <Step title="Customize via CSS classes">
    Add custom classes to PrimeReact components and style them in App.css rather than inline styles.
  </Step>
</Steps>
