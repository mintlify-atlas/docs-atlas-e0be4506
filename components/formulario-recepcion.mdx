## Overview

The `FormularioRecepcion` component provides a form interface for receiving vehicles or units. This component is currently in early development and displays a placeholder.

## Planned Features

Based on the application structure and the similar `FormularioDespacho` component, the reception form should include:

### Form Fields

- **Modulo (Module)**: Dropdown to select receiving module
- **Economico (Economic Number)**: Text input for vehicle identification
- **Estado (Condition)**: Dropdown or checklist for vehicle condition
- **Observaciones (Observations)**: Text area for additional notes
- **Hora de Recepción (Reception Time)**: Timestamp or time picker

### UI Components

Expected PrimeReact components:

- **TabView/TabPanel**: Tabbed interface for organization
- **Dropdown**: Select inputs for modules and conditions
- **InputText**: Text inputs for vehicle information
- **InputTextarea**: Multi-line input for observations
- **Calendar**: Date/time picker for reception time
- **Button**: Submit and clear actions
- **Card**: Container for form layout

### State Management

## Expected Implementation

Based on the pattern from `FormularioDespacho`, the component should follow this structure:

```tsx
import { TabView, TabPanel } from "primereact/tabview";
import { InputText } from "primereact/inputtext";
import { InputTextarea } from "primereact/inputtextarea";
import { Dropdown } from "primereact/dropdown";
import { Calendar } from "primereact/calendar";
import { Button } from "primereact/button";
import { useState } from "react";
import { useHook_General } from "../hooks/useHook";

export const FormularioRecepcion = () => {
  const { modulosOptions, estadosOptions } = useHook_General();
  
  const [formData, setFormData] = useState({
    modulo: null,
    economico: "",
    estado: null,
    observaciones: "",
    horaRecepcion: new Date()
  });

  const handleSubmit = () => {
    // Validate and submit form
  };

  const handleClear = () => {
    setFormData({
      modulo: null,
      economico: "",
      estado: null,
      observaciones: "",
      horaRecepcion: new Date()
    });
  };

  return (
    <TabView>
      <TabPanel className="tabpanel" header="Recepción">
        <div className="d-flex justify-content-center">
          <div className="card">
            <div className="titulo">
              <h1>Formulario de recepción</h1>
              <hr />
            </div>

            <div className="d-flex flex-column gap-3 mt-4 py-2 p-4">
              {/* Modulo */}
              <span className="p-float-label">
                <Dropdown
                  inputId="modulo"
                  value={formData.modulo}
                  onChange={(e) => setFormData({...formData, modulo: e.value})}
                  options={modulosOptions}
                  className="select"
                />
                <label htmlFor="modulo">Módulo</label>
              </span>

              {/* Economico */}
              <span className="p-float-label">
                <InputText
                  id="economico"
                  value={formData.economico}
                  onChange={(e) => setFormData({...formData, economico: e.target.value})}
                  className="select"
                />
                <label htmlFor="economico">Económico</label>
              </span>

              {/* Estado */}
              <span className="p-float-label">
                <Dropdown
                  inputId="estado"
                  value={formData.estado}
                  onChange={(e) => setFormData({...formData, estado: e.value})}
                  options={estadosOptions}
                  className="select"
                />
                <label htmlFor="estado">Estado</label>
              </span>

              {/* Hora de Recepción */}
              <span className="p-float-label">
                <Calendar
                  inputId="hora"
                  value={formData.horaRecepcion}
                  onChange={(e) => setFormData({...formData, horaRecepcion: e.value})}
                  showTime
                  hourFormat="24"
                />
                <label htmlFor="hora">Hora de Recepción</label>
              </span>

              {/* Observaciones */}
              <span className="p-float-label">
                <InputTextarea
                  id="observaciones"
                  value={formData.observaciones}
                  onChange={(e) => setFormData({...formData, observaciones: e.target.value})}
                  rows={5}
                  className="w-100"
                />
                <label htmlFor="observaciones">Observaciones</label>
              </span>
            </div>

            <div className="d-flex justify-content-center gap-3 mt-4">
              <Button 
                icon="pi pi-check" 
                label="Recibir" 
                severity="success"
                onClick={handleSubmit}
              />
              <Button 
                icon="pi pi-times" 
                label="Limpiar" 
                severity="danger"
                onClick={handleClear}
              />
            </div>
          </div>
        </div>
      </TabPanel>
    </TabView>
  );
};
```

## Validation Requirements

Recommended validation rules:

```tsx
const validateForm = () => {
  const errors = [];
  
  if (!formData.modulo) {
    errors.push("Módulo es requerido");
  }
  
  if (!formData.economico || formData.economico.trim() === "") {
    errors.push("Económico es requerido");
  }
  
  if (!formData.estado) {
    errors.push("Estado es requerido");
  }
  
  if (!formData.horaRecepcion) {
    errors.push("Hora de recepción es requerida");
  }
  
  return errors;
};
```
